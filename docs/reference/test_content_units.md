# Test for content unit links

Test for content unit links

## Usage

``` r
test_content_units(metadata = load_metadata(directory))
```

## Arguments

- metadata:

  The metadata object returned by `load_metadata`. If parameter is not
  provided, it defaults to calling `load_metadata` in the current
  project directory.

## Value

invisible(metadata)

## Examples

``` r
if (FALSE) { # \dontrun{
test_content_units()
} # }
```
